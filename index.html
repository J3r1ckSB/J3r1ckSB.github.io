Preguntas frecuentes

1. ¿Cuánto costará mi sitio web?

Depende del tipo de sitio web, las funcionalidades y extras que requieras. El precio mínimo para las páginas informativas, tiendas en línea y páginas corporativas son 499.990/2.099.990/1.399.990 COP respectivamente. Para más información, visita nuestra Políticas de precios.

2. ¿Cuánto tiempo tardará mi sitio web en ser entregado?

De la misma forma; depende del tipo de sitio web y todo el trabajo que se requiera hacer.
Como referencia; una página informativa común puede tardar hasta 5 días hábiles con entrega normal y 2 días hábiles con servicio Sitio Web Express.
Una tienda en línea común puede tardar hasta 14 días hábiles con entrega normal y de 3 a 7 días hábiles con servicio Sitio Web Express.
Por último, una página corporativa común puede tardar hasta 14 días hábiles con entrega normal y de 3 a 6 días hábiles con servicio Sitio Web Express.

3. ¿Cómo puedo confiar de ustedes?

En e-Maker aceptamos métodos de pago reconocidos con protocolos de protección al comprador como MercadoPago® y Paypal®. Además, todo nuestro personal está publicado en la sección de Quiénes Somos con sus respectivas redes sociales y métodos de contacto.

4. ¿Cómo puedo confiar en que me entregarán un sitio web que cumpla con mis expectativas?

En e-Maker siempre velamos por el bien del usuario, por tal motivo, nos comprometemos a entregar un sitio web que cumpla con las expectativas del usuario o le devolvemos el dinero. En caso de enseñar el prototipo del sitio web al usuario, y este no ser como lo deseaba el usuario, ofrecemos la opción de rehacerlo hasta que cumpla con los requerimientos del usuario o le devolvamos la totalidad del dinero.
Por otro lado, ofrecemos garantía de por vida ante cualquier fallo que pueda presentar un sitio web realizado por nosotros que sea causado por error nuestro.
Para más información, ingresa a nuestra sección de Política de Garantía.

5. ¿Puedo pedir un sitio web si no tengo claro qué diseño quiero?

Sin ningún tipo de problema. Nosotros te propondremos diseños acorde a tus necesidades y te enseñaremos (al cabo de unos días) un prototipo del sitio web. También puedes echarle una vistazo a nuestro portafolio de diseños.
Además, ofrecemos diseño de logos e identidad de marca en caso de que lo necesites. Para más información, ingresa a la sección de Políticas de diseño.

6. ¿Cómo puedo modificar mi sitio web después de entregado?

e-Maker proveerá al usuario de información para que puedan modificar sus sitios web. En caso de que lo necesites, podemos modificar el sitio web por ti. Si necesitas modificar, añadir o eliminar textos, imágenes y/o vídeos; te ofrecemos las 10 primeras modificaciones, adiciones o eliminaciones totalmente gratis o 1 mes gratis (lo que ocurra primero). Para modificaciones mayores, contáctate con nosotros

Diseño y experiencia de usuario

e-Maker cuenta con templates y diseñadores web del más alto nivel. Tu sitio web gozará de un diseño minimalista, moderno, intuitivo y único. ¡Cautivarás a tus clientes desde el primer momento en el que ingresen a tu sitio web!

Optimización y rendimiento

Nuestros sitios web están preparados para funcionar de manera óptima en cualquier dispositivo del mercado. Además, usamos técnicas de optimización de código para que los sitios web carguen mucho más rápido que los de la competencia.

Estándares de seguridad

En e-Maker desarrollamos todos los sitios web implementando los últimos estándares y protocolos de seguridad existentes. Además, todos los sitios web desarrollados por nosotros incluyen protección en contra de ataques DDOS, entre otros; por medio de CloudFlare® sin costo adicional alguno (sólo aplica si el hosting se realiza a través de Siteground durante el proceso de compra).

Implementaciones/Integraciones

Al momento de pedir tu sitio web, te daremos la posibilidad de añadir diferentes integraciones en tu sitio web sin costo adicional; entre las cuales se encuentran:

- Feed de Facebook y/o Twitter
- Facebook Shop y/o Instagram Store
- Google Ads (publicita tu sitio web en internet)
- Google AdSense (obtén ingresos mostrando publicidad en tu sitio web)
- Google Analytics (obtén estadísticas de uso detalladas sobre tu sitio web)
- Aplicación para administrar los pedidos de tu sitio web desde tu smartphone (sólo disponible para tiendas en línea)
- Entre otras (disponibles al momento de la compra)

En caso de que requieras de una integración que no ofrezcamos, la podremos realizar por un bajo costo adicional.

Aplicación web incluída (PWA)

Todo usuario tendrá derecho a recibir una aplicación web de su sitio web compatible con todo tipo de smartphones, PCs, etc.; sin costo adicional. Un ejemplo de aplicación web es la aplicación para escritorio de WhatsApp®, disponible en la Microsoft Store®; la cual es una aplicación web del sitio web https://web.whatsapp.com.
